How this simulator works

Text is embedded using weighted word-frequency vectors (a simplified stand-in for a real neural embedding model), so results are driven by real keyword/semantic overlap between your query and documents. Similarity is true cosine similarity; the scatter plot is a random 2D projection of that same vector, just for visualization.

Built as an educational RAG simulator — all computation runs locally in your browser.
